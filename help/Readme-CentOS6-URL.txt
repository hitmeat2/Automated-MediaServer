Install CentOS 6
	
	---Load from URL----
					
					Press Tab at the Install or Upgrade line and type "text" after initrd.img and press enter for a CLI install
					
					Skip "Test Media" and begin installation
					
					Select Languge, Country, ectcetera.
					
					Choose "Re-Initialize"
					
					Select Time Zone
					
					Set Password to "mediaeerver" then type "mediaserver" again to confirm.
					
					Select use entire disk. Make sure it is highlighted then move down to OK.
					
					To Confirm, Choose "Write Changes to Disk"
					
					**AT this time the system will format your disk and prepare the CENTOS6 for startup.
					
					Clic "Reboot" to restart the system.
